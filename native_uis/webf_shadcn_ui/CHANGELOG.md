# Changelog

## 0.1.0

- Initial release
- Added theming support with 12 color schemes and light/dark mode
- Added Form Controls: Button, Input, Textarea, Checkbox, Radio, Switch, Select, Slider, Combobox, Form
- Added Display Components: Card, Alert, Badge, Avatar, Toast, Tooltip, Progress, Separator
- Added Navigation/Layout: Tabs, Dialog, Sheet, Popover, Breadcrumb, DropdownMenu, ContextMenu
- Added Data Display: Table, Accordion, Calendar, DatePicker, TimePicker, Image
- Added Advanced Components: ScrollArea, Skeleton, Collapsible
