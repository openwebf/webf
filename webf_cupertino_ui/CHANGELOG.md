# Changelog

## 0.0.1

* Initial release
* Migrated all Cupertino UI components from WebF core to a standalone package
* Added support for all major iOS-style UI components:
  - Buttons, inputs, switches, sliders, checkboxes, radio buttons
  - Pickers (standard, date, timer)
  - Dialogs (alerts, action sheets, context menus)
  - Navigation (tabs, segmented controls)
  - Form components (form rows, form sections)
  - List components (list sections, list tiles)
  - Other components (icons, loading indicators, toasts, search inputs, text areas)
* Included TypeScript definitions for all components
* Full integration with WebF's WidgetElement system
